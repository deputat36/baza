# Обучение сотрудников работе с базой

## Назначение

План обучения фиксирует, что должны уметь ключевые роли перед запуском базы знаний в офисе.

Он нужен, чтобы запуск не свёлся к фразе «таблица где-то лежит». Каждый сотрудник должен понимать свой маршрут, где искать информацию и как предлагать исправления.

## Основной файл

`data/drafts/role-training-plan.csv`

## Отчет

```bash
python scripts/tools/build_role_training_report.py
```

Результат:

- `build/role-training-report.md`
- `build/role-training-report.csv`

## Как использовать

1. Открыть отчет.
2. Провести короткое обучение по ролям.
3. Дать каждому сотруднику практическое задание.
4. Отметить в закрытой рабочей таблице, кто прошел обучение.
5. Блокирующие блоки перевести в `READY` или `DONE` только после фактической проверки.

## Важно

В публичном репозитории хранится только структура обучения.

ФИО сотрудников, результаты прохождения и внутренние комментарии должны храниться в закрытой Google Таблице.

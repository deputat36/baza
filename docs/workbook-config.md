# Конфигурация XLSX-сборки

## Назначение

Файл `scripts/tools/workbook_config.py` хранит общую карту XLSX-сборки:

- название выходного файла;
- каталог сборки;
- список листов;
- CSV-источники для каждого листа.

## Почему это отдельный файл

Раньше карта листов находилась внутри `build_workbook_from_csv.py`.

Из-за этого проверка источников импортировала XLSX-генератор и зависела от `openpyxl`, хотя для проверки наличия CSV эта библиотека не нужна.

Теперь:

- `build_workbook_from_csv.py` отвечает за создание XLSX;
- `validate_workbook_sources.py` отвечает за проверку источников;
- `workbook_config.py` хранит общие настройки без внешних зависимостей.

## Как добавить CSV в XLSX

1. Открыть `scripts/tools/workbook_config.py`.
2. Найти нужный лист в `SHEETS`.
3. Добавить путь к CSV.
4. Запустить `make sources`.
5. Запустить `make xlsx` или `make preflight`.

## Важно

Не добавляйте в конфигурацию приватные CSV с реальными контактами, данными клиентов или внутренними условиями.

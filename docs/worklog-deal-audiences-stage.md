# Этап аудиторий подсказок

## Добавлено

- Словарь аудиторий будущих подсказок сделки.
- Проверка поля `Кому показывать` в правилах подсказок.
- Отчёт покрытия аудиторий правилами и сценариями.
- Подключение отчёта к Makefile, GitHub Actions и Pages-сборке.

## Практический смысл

Будущий интерфейс сможет различать, кому показывать подсказку: СПН, юристу, менеджеру, брокеру или администратору базы.

## Важно

Проект Навигатора сделок не менялся.

# Чек-лист публичной проверки

Перед публикацией изменений нужно проверить:

- нет ли реальных рабочих телефонов;
- нет ли данных клиентов;
- нет ли закрытых условий;
- нет ли ключей, паролей и токенов;
- нет ли приватных комментариев по людям;
- все примеры обезличены;
- реальные контакты оставлены только для закрытой Google Таблицы.

Если есть сомнение, файл не публиковать до ручной проверки.

# Маршрутизация в департаменты центрального офиса

## Назначение

Раздел нужен менеджеру офиса Борисоглебска, чтобы понимать, куда направлять вопросы по центральному офису: юридические, ипотечные, обучающие, IT, маркетинговые, финансовые и другие.

## Важно

Публичный репозиторий хранит только структуру маршрутизации.

Реальные ФИО кураторов, телефоны, почты, чаты и внутренние каналы должны храниться только в закрытой Google Таблице.

## Исходный файл

`data/drafts/central-office-routing.csv`

## Генератор

`scripts/tools/build_department_routing_report.py`

## Результаты

- `build/department-routing-report.md`
- `build/department-routing-report.csv`

## Запуск

```bash
make department-routing
```

Или в составе полной проверки:

```bash
make preflight
```

## Что заполнить в закрытой таблице

- ФИО куратора или представителя департамента.
- Рабочий канал связи.
- Замещающий контакт.
- Правила эскалации.
- Ограничения по передаче документов и персональных данных.

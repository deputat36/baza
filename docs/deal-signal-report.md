# Отчёт по сигналам сделки

## Назначение

Отчёт показывает, насколько словарь сигналов сделки покрыт правилами подсказок.

Это помогает развивать будущую совместимость без подключения к Навигатору сделок.

## Источники

```text
data/dictionaries/deal-signals.csv
data/drafts/deal-hint-rules.csv
```

## Генератор

```text
scripts/tools/build_deal_signal_report.py
```

## Артефакты

```text
build/deal-signal-report.md
build/deal-signal-report.csv
```

## Что проверяется

- какие сигналы есть в словаре;
- какие сигналы уже используются в правилах подсказок;
- какие сигналы пока не покрыты правилами;
- есть ли в правилах неизвестные сигналы.

## Команда

```bash
make deal-signals
```

Или полный набор:

```bash
make preflight
```

# Меню `База знаний` в Google Таблице

## Назначение

Файл `scripts/google-apps-script/menu.gs` добавляет в закрытую Google Таблицу пользовательское меню `База знаний`.

Меню нужно, чтобы запускать настройку и проверку таблицы без перехода в редактор Apps Script.

## Пункты меню

```text
База знаний
├─ 1. Создать базовые листы
├─ 2. Применить оформление
├─ 3. Проверить данные
└─ Полная подготовка таблицы
```

## Что должно быть подключено

В проекте Apps Script должны быть файлы:

```text
setup-baza.gs
setup_baza_sheet.gs
validate-baza-sheet.gs
menu.gs
```

## Как включить меню

1. Открыть закрытую Google Таблицу.
2. Перейти в `Расширения -> Apps Script`.
3. Добавить файл `menu.gs`.
4. Сохранить проект.
5. Обновить страницу Google Таблицы.
6. В верхнем меню появится пункт `База знаний`.

## Первый запуск

При первом запуске Google попросит разрешения.

После выдачи разрешений нужно вернуться в таблицу и повторить действие через меню.

## Рекомендуемый порядок

1. `Полная подготовка таблицы` — для первичной сборки.
2. Импорт CSV или ручное заполнение данных.
3. `3. Проверить данные` — после заполнения.
4. Исправить замечания на листе `Проверка данных`.
5. Повторить проверку.

## Ограничение

Меню не импортирует реальные контакты и не проверяет актуальность информации.

Оно только запускает технические функции настройки и проверки закрытой Google Таблицы.

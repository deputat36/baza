# Сигналы сделки

## Назначение

Сигналы сделки — это безопасный словарь признаков, по которым в будущем можно будет показывать подсказки базы знаний в карточке сделки.

Сейчас этот словарь не подключён к Навигатору сделок и не меняет его поведение.

## Источник

```text
data/dictionaries/deal-signals.csv
```

## Где используется

Словарь используется генератором правил подсказок:

```text
scripts/tools/build_deal_hint_rules.py
```

Если в `data/drafts/deal-hint-rules.csv` указан сигнал, которого нет в словаре, сборка правил должна завершиться ошибкой.

## Примеры сигналов

- `has_children`
- `has_mortgage`
- `has_matcap`
- `document_needed`
- `risk_signal`
- `stage`
- `client_question`
- `precheck`

## Правило безопасности

Сигналы не должны содержать персональные данные, ФИО клиентов, телефоны, адреса объектов и комментарии по конкретным сделкам.

Они описывают только тип признака, а не саму сделку.

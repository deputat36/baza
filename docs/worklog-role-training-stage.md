# Этап обучения сотрудников

## Добавлено

- CSV с планом обучения по ролям.
- Генератор отчета по обучению.
- Инструкция по проведению обучения.

## Практический смысл

Перед запуском сотрудники должны не просто получить ссылку на базу, а пройти короткие практические сценарии по своей роли.

## Следующий шаг

Собрать отчет, провести обучение и отметить фактическое прохождение в закрытой Google Таблице.

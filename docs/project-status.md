# Статус проекта

Дата обновления: 25.06.2026

## Текущий этап

Проект находится на этапе архитектурного MVP.

Уже создана структура базы знаний, шаблоны данных, черновики основных разделов и документы для запуска первой закрытой Google Таблицы.

Проект ещё не является финальной рабочей базой, потому что реальные контакты, внутренние телефоны и проверенные городские данные должны заполняться в закрытой таблице.

## Что нужно сделать следующим шагом

1. Создать закрытую Google Таблицу.
2. Импортировать первые CSV-файлы.
3. Настроить листы, фильтры, статусы и приоритеты.
4. Заполнить реальные контакты только в закрытой таблице.
5. Назначить ответственных за разделы.
6. Провести первый пользовательский тест.
7. По итогам теста упростить структуру и убрать лишние поля.

## Что нельзя делать до приватизации

Нельзя добавлять в публичный репозиторий:

- реальные внутренние телефоны;
- личные контакты партнёров;
- ФИО клиентов;
- данные сделок;
- закрытые условия компании;
- документы клиентов;
- рабочие комментарии, которые не предназначены для публичного доступа.

## Критерий перехода к следующему этапу

Следующий этап начинается после того, как 2–3 сотрудника протестируют Google Таблицу и подтвердят, что могут найти нужный ответ за 1–3 минуты.

После этого проект можно переводить из архитектурного MVP в рабочий MVP офиса.

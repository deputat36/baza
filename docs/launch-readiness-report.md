# Отчёт готовности к запуску

## Назначение

Отчёт показывает техническую готовность листов будущей закрытой Google Таблицы.

Генератор:

```text
scripts/tools/build_launch_readiness_report.py
```

Результаты:

```text
build/launch-readiness-report.md
build/launch-readiness-report.csv
```

## Запуск

Через Makefile:

```bash
make readiness
```

Или напрямую:

```bash
python scripts/tools/build_launch_readiness_report.py
```

В составе полной проверки:

```bash
make preflight
```

## Статусы

`READY` — источники есть, строки есть, пустых ID, статусов и приоритетов не найдено.

`CHECK` — раздел технически есть, но перед переносом нужна ручная проверка обязательных полей.

`DRAFT` — нет строк данных или отсутствует источник.

## Как использовать

1. Открыть `build/launch-readiness-report.md`.
2. Сначала разобрать разделы со статусом `DRAFT`.
3. Затем исправить или вручную проверить разделы `CHECK`.
4. После этого сверить `missing-values-report.md`.
5. Только потом переносить данные в закрытую Google Таблицу.

## Ограничение

Отчёт оценивает только техническую готовность.

Он не подтверждает актуальность контактов, юридическую точность и полноту рабочих комментариев.

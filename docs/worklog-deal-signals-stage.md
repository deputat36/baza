# Этап словаря сигналов сделки

## Добавлено

- Словарь сигналов сделки.
- Проверка сигналов в правилах подсказок.
- Отчёт по покрытию сигналов правилами.
- Документация по сигналам и отчёту сигналов.

## Принцип

Навигатор сделок не меняется. `baza` получает контролируемый словарь признаков, который можно будет использовать в будущей read-only интеграции.

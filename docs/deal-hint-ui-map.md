# UI-карта подсказок сделки

## Назначение

UI-карта описывает, где в будущем интерфейсе сделки должна отображаться каждая подсказка.

Проект Навигатора сделок сейчас не меняется.

## Источники

- `data/dictionaries/deal-hint-ui-zones.csv` — допустимые зоны интерфейса.
- `data/drafts/deal-hint-ui-map.csv` — связь правила подсказки с зоной UI.
- `build/deal-hint-rules.json` — правила подсказок, которые проверяются по карте.

## Команда

```bash
make deal-hint-ui-map
```

Полная проверка:

```bash
make preflight
```

## Артефакты

- `build/deal-hint-ui-map.json`
- `build/deal-hint-ui-map-report.md`
- `build/deal-hint-ui-map-report.csv`

## Практический смысл

Будущий интерфейс сможет заранее понимать, где показывать подсказку:

- в предупреждении о риске;
- в чек-листе документов;
- в боковой панели;
- в контрольном блоке менеджера;
- в ипотечном блоке;
- в чек-листе передачи объекта.

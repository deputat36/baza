# Навигация по ролям

## Назначение

Навигация по ролям показывает, с каких разделов и отчётов начинать разным сотрудникам.

Это нужно, чтобы база была не просто набором файлов, а рабочим маршрутом для СПН, менеджера, юриста, брокера и администратора базы.

## Файл данных

`data/drafts/role-navigation-map.csv`

## Команда

```bash
make role-navigation
```

Обычно отдельно запускать не нужно: отчёт входит в `make preflight`.

## Результаты

Скрипт создаёт:

- `build/role-navigation-report.md`;
- `build/role-navigation-report.csv`.

## Что проверяется

- роль есть в `data/dictionaries/deal-audiences.csv`;
- заполнены сценарий, стартовая точка, разделы, артефакты и следующее действие;
- приоритет указан от 1 до 5;
- ID не дублируются.

## Важно

Отсутствующие build-артефакты фиксируются как состояние `missing_now`, но не ломают CI. Некоторые файлы появляются только после полной сборки.

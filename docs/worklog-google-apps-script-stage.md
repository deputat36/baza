# Этап Google Apps Script

## Добавлено

- Отдельный помощник Apps Script для оформления закрытой Google Таблицы.
- Инструкция по использованию базового и дополнительного Apps Script.
- Уточнение индекса проекта по Apps Script-файлам.

## Файлы

```text
scripts/google-apps-script/setup-baza.gs
scripts/google-apps-script/setup_baza_sheet.gs
docs/google-apps-script-setup.md
```

## Практический смысл

Закрытую Google Таблицу можно быстрее подготовить к работе: создать базовую структуру, затем применить оформление, фильтры, выпадающие списки и условное форматирование.

## Порядок запуска

1. Запустить `setupBazaKnowledgeSheet`.
2. Запустить `applyBazaSheetFormattingPlan`.
3. Проверить листы, фильтры, статусы, приоритеты и цвета.

## Ограничение

Скрипты не импортируют реальные контакты и не заменяют ручную проверку данных.

Реальные рабочие данные остаются только в закрытой Google Таблице.

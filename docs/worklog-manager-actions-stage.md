# Этап реестра управленческих действий

## Добавлено

- Генератор реестра управленческих действий.
- Документация по реестру действий.
- Цель `make manager-actions`.
- Сборка реестра в GitHub Actions и GitHub Pages workflow.
- Реестр действий в пакете запуска, preflight-сводке и стартовом индексе артефактов.
- Еженедельная задача в регламенте эксплуатации.

## Практический смысл

Менеджер получает не только обзор состояния проекта, но и список конкретных открытых действий, которые нужно назначить ответственным.

## Команда

```bash
make manager-actions
```

## Результаты

- `build/manager-action-register.md`
- `build/manager-action-register.csv`

## Следующий шаг

Запустить сборку, открыть реестр действий и перенести реальные задачи в закрытый рабочий контур.

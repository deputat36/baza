# Preflight-сводка

## Назначение

Скрипт `scripts/tools/build_preflight_summary.py` создаёт короткую сводку по результатам технической проверки.

Он собирает в одном файле:

- список ожидаемых артефактов;
- наличие каждого файла в `build/`;
- ручной чек-лист перед импортом;
- критерий технической готовности.

## Запуск

```bash
python scripts/tools/build_preflight_summary.py
```

Через Makefile:

```bash
make summary
```

## Результат

```text
build/preflight-summary.md
```

## Как использовать

После `make preflight` или запуска GitHub Actions нужно открыть `preflight-summary.md` первым.

Он покажет, какие файлы собраны и какие ручные проверки нужно пройти перед переносом данных в закрытую Google Таблицу.

## Ограничение

Сводка не подтверждает актуальность контактов и юридическую точность данных. Она фиксирует только техническую готовность структуры.

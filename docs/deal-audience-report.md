# Отчёт по аудиториям подсказок

## Назначение

Отчёт показывает, какие роли используются в правилах подсказок и безопасных тестовых сценариях.

Он помогает понять:

- какие аудитории уже покрыты правилами;
- какие роли указаны в сценариях;
- есть ли неизвестные роли, которых нет в словаре;
- какие роли из словаря пока не используются.

## Команда

```bash
make deal-audiences
```

## Артефакты

- `build/deal-audience-report.md`
- `build/deal-audience-report.csv`

## Ограничение

Отчёт проверяет только структуру и согласованность ролей. Он не решает, какие подсказки действительно нужны конкретному сотруднику в рабочем процессе.

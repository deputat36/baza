# Этап приемочных сценариев офиса

## Добавлено

- CSV со сценариями приемки базы для ролей офиса.
- Генератор отчета по приемочным сценариям.
- Инструкция по ручной приемке.

## Практический смысл

Теперь запуск базы можно проверять не только технически, но и по реальным рабочим действиям сотрудников.

## Следующий шаг

Собрать отчет, пройти блокирующие сценарии вручную и перенести фактические результаты в закрытую Google Таблицу.

# Этап плана оформления Google Таблицы

## Добавлено

- Генератор плана оформления закрытой Google Таблицы.
- Документ с описанием генерируемого плана оформления.
- Makefile-команда `formatting-plan`.
- Шаг GitHub Actions для сборки плана оформления.
- Учёт артефактов оформления в preflight-сводке.
- Обновление инструкций по preflight, CI, отчётам, макету таблицы и индексу проекта.

## Практический смысл

Перед ручной сборкой закрытой Google Таблицы теперь есть отдельный артефакт с правилами оформления, фильтров, закрепления строк и условного форматирования.

## Основная команда

```bash
make formatting-plan
```

## Общая проверка

```bash
make preflight
```

## Ожидаемые артефакты

```text
build/google-sheet-formatting-plan.md
build/google-sheet-formatting-plan.csv
```

## Следующий шаг

Собрать артефакты через GitHub Actions или локально, затем использовать `google-sheet-formatting-plan.md` при настройке закрытой Google Таблицы.

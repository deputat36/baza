# Аудитории подсказок сделки

## Назначение

`data/dictionaries/deal-audiences.csv` задаёт контролируемый список ролей, которым могут показываться будущие подсказки в карточке сделки.

Это нужно, чтобы поле `Кому показывать` в правилах подсказок не было произвольным текстом.

## Сейчас

Проект не подключается к Навигатору сделок и не меняет его код.

Словарь нужен только как подготовка контракта данных для будущего интерфейса.

## Роли

- `СПН` — специалист по недвижимости.
- `юрист` — юридическая проверка рисков и документов.
- `менеджер` — контроль качества сделки и работы сотрудников.
- `брокер` — ипотека, банки, маткапитал и расчёты.
- `администратор базы` — сопровождение структуры и качества базы знаний.

## Проверка

Роли проверяются в правилах подсказок и тестовых сценариях.

```bash
make deal-audiences
```

Если роль не найдена в словаре, отчёт завершится ошибкой.

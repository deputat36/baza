# Этап конфигурации XLSX-сборки

## Добавлено

- `scripts/tools/workbook_config.py` с общей картой XLSX-сборки.
- Документация `docs/workbook-config.md`.
- `build_workbook_from_csv.py` теперь использует общий конфиг.
- `validate_workbook_sources.py` больше не импортирует XLSX-генератор и не зависит от `openpyxl`.
- Обновлены инструкции по XLSX, проверке источников и индекс проекта.

## Практический смысл

Состав листов XLSX теперь меняется в одном месте, а лёгкая проверка источников может работать без Excel-зависимостей.

## Следующий шаг

Запустить `make sources` перед `make install`, если нужно быстро проверить только наличие и читаемость CSV-источников.

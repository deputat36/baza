# Этап меню Google Таблицы

## Добавлено

- Apps Script-файл `menu.gs`.
- Пользовательское меню `База знаний` в закрытой Google Таблице.
- Отдельная инструкция по меню.
- Обновление общей инструкции Apps Script.
- Обновление чек-листа запуска Google Таблицы.
- Обновление индекса проекта.

## Файлы

```text
scripts/google-apps-script/menu.gs
docs/google-apps-script-menu.md
docs/google-apps-script-setup.md
docs/google-sheet-launch-checklist.md
docs/project-index.md
```

## Пункты меню

```text
1. Создать базовые листы
2. Применить оформление
3. Проверить данные
Полная подготовка таблицы
```

## Практический смысл

Менеджеру не нужно каждый раз открывать редактор Apps Script для запуска функций.

После подключения скриптов основные действия выполняются из меню закрытой Google Таблицы.

## Следующий шаг

Подключить четыре Apps Script-файла в закрытую Google Таблицу, обновить страницу и проверить появление меню `База знаний`.

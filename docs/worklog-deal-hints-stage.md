# Этап правил подсказок сделки

## Добавлено

- Черновик правил подсказок сделки.
- Генератор JSON-контракта правил.
- Отчёт по правилам подсказок.
- Документация по правилам.

## Принцип

Навигатор сделок не меняется. `baza` получает подготовительный слой, который в будущем позволит показывать релевантные подсказки в карточке сделки.

## Следующий шаг

Расширять правила по мере появления новых ситуаций, документов и локальной практики офиса.

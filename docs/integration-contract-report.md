# Отчёт по контрактам интеграции

## Назначение

Отчёт проверяет, что для каждого будущего контракта интеграции есть источник, артефакт сборки, документация и генератор.

## Команда

```bash
make integration-contracts
```

## Артефакты

- `build/integration-contract-report.md`
- `build/integration-contract-report.csv`

## Что означает ошибка

Ошибка означает, что контракт описан в реестре, но один из его элементов отсутствует.

Такой контракт нельзя считать готовым для будущего подключения.

# Этап матрицы доступа интеграции

## Добавлено

- Черновик матрицы доступа ролей к областям данных.
- Скрипт проверки матрицы доступа.
- Markdown и CSV отчёты по доступам.
- Документация по матрице доступа.
- Подключение проверки к preflight, GitHub Actions, Pages-сборке, XLSX и индексу артефактов.

## Практический смысл

Проект можно развивать в сторону будущей интеграции без смешивания публичных, внутренних и клиентских данных.

## Ограничение

Навигатор сделок не изменялся и не подключался.

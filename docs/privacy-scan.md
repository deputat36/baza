# Проверка публичной безопасности

## Назначение

Скрипт `scripts/tools/privacy_scan.py` помогает найти в репозитории потенциально рискованные данные перед публикацией.

Он ищет:

- похожие на телефоны строки;
- похожие на email строки;
- слова, связанные с доступами и персональными данными.

## Запуск

```bash
python scripts/tools/privacy_scan.py
```

## Важно

Скрипт не принимает решение за человека.

Если он нашёл замечание, нужно вручную проверить файл и понять, можно ли оставлять эти данные в публичном репозитории.

## Главное правило

Реальные рабочие контакты, данные клиентов и закрытые условия должны храниться только в закрытой Google Таблице.

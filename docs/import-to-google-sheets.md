# Импорт в Google Таблицу

## Назначение

GitHub хранит структуру проекта и шаблоны. Google Таблица используется как рабочий инструмент сотрудников.

## Первый набор для импорта

1. `templates/contacts-template.csv`
2. `templates/knowledge-template.csv`
3. `templates/situations-template.csv`
4. `data/drafts/government-borisoglebsk.csv`
5. `data/drafts/documents-real-estate.csv`
6. `data/drafts/situations-real-estate.csv`
7. `data/drafts/banks-mortgage.csv`
8. `data/drafts/notaries-structure.csv`
9. `data/drafts/cadastral-engineers.csv`

## Рекомендуемые листы

- Главная
- Контакты
- База знаний
- Ситуации
- Практика
- Предложения изменений
- Справочники

## Базовые настройки

- закрепить первую строку;
- включить фильтр;
- добавить выпадающий список для статуса;
- добавить выпадающий список для приоритета;
- использовать справочник категорий;
- выделять строки со статусом `Нужно проверить`;
- отдельно хранить предложения сотрудников.

## Минимальный запуск

Для первого запуска достаточно трёх листов:

1. Контакты.
2. Ситуации.
3. Практика.

После этого сотрудники уже смогут пользоваться базой и предлагать улучшения.

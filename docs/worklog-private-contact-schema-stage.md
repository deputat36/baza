# Этап схемы закрытого реестра контактов

## Добавлено

- CSV-схема закрытого реестра контактов.
- Генератор отчета по схеме закрытых контактов.
- Документация по закрытому реестру контактов.
- Цель `make private-contact-schema`.
- Сборка отчета в GitHub Actions и GitHub Pages workflow.
- Отчет в пакете запуска, preflight-сводке и стартовом индексе артефактов.
- Ежемесячная задача в регламенте эксплуатации.

## Практический смысл

Проект теперь описывает, какие поля нужно заполнить в закрытой Google Таблице для кураторов, департаментов, городских служб, подрядчиков, нотариусов и кадастровых инженеров.

## Безопасность

Публичный репозиторий хранит только схему. ФИО, телефоны, почты, чаты, замещающие контакты и комментарии по людям должны храниться только в закрытом рабочем контуре.

## Команда

```bash
make private-contact-schema
```

## Результаты

- `build/private-contact-schema-report.md`
- `build/private-contact-schema-report.csv`

# Реестр ID

## Назначение

Скрипт `scripts/tools/build_id_registry_report.py` собирает ID из первого столбца CSV-файлов проекта и формирует отчёт.

Он помогает увидеть:

- сколько ID есть в данных;
- какие префиксы используются;
- есть ли возможные дубли;
- где находится конкретный ID;
- какие строки не удалось распознать как ID.

## Запуск

```bash
python scripts/tools/build_id_registry_report.py
```

Через Makefile:

```bash
make ids
```

## Результат

```text
build/id-registry.md
```

## Как использовать

Перед импортом в закрытую Google Таблицу нужно открыть отчёт и проверить раздел `Возможные дубли`.

Если ID повторяется случайно, одну из записей нужно переименовать.

Если повтор связан с шаблоном или служебной строкой, это нужно учитывать вручную.

## Ограничение

Отчёт диагностический и не блокирует workflow сам по себе.

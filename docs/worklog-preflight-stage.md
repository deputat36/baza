# Этап preflight-проверок

## Добавлено

- Скрипт проверки публичной безопасности.
- Инструкция по privacy scan.
- Makefile с командами проекта.
- Preflight-инструкция.
- Обновление GitHub Actions workflow.
- Чек-лист публичной проверки.

## Практический смысл

Перед переносом данных и перед публикацией изменений теперь можно запускать единый набор проверок.

## Команда

```bash
make preflight
```

## Следующий шаг

Запустить preflight, исправить замечания и собрать XLSX для просмотра структуры.

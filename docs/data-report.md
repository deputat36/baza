# Отчёт по данным

## Назначение

Скрипт `scripts/tools/build_data_report.py` собирает краткую техническую сводку по CSV-файлам проекта.

Он показывает:

- количество CSV-файлов;
- количество строк данных;
- количество колонок в каждом файле;
- распределение статусов, если в файле есть колонка `Статус` или `status`.

## Запуск

```bash
python scripts/tools/build_data_report.py
```

Результат сохраняется здесь:

```text
build/data-report.md
```

## Использование

Отчёт нужен перед переносом данных в закрытую Google Таблицу и после крупных изменений в CSV.

Он помогает быстро увидеть, какие разделы уже наполнены, а какие ещё пустые или слабые.

## Ограничение

Отчёт технический. Он не подтверждает актуальность контактов, юридическую точность и полноту данных.

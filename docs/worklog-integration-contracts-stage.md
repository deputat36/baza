# Этап контрактов интеграции

## Добавлено

- Реестр контрактов будущей интеграции.
- Генератор отчёта по контрактам.
- Документация по контрактам и отчёту.
- Подключение проверки контрактов к Makefile, GitHub Actions и Pages-сборке.

## Практический смысл

Будущее подключение к другому интерфейсу можно будет строить от явно описанных read-only артефактов, а не от неформальных договорённостей.

## Важно

Проект Навигатора сделок не менялся.

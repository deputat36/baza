# Этап проверки источников XLSX

## Добавлено

- Скрипт `scripts/tools/validate_workbook_sources.py`.
- Документация `docs/workbook-sources-validation.md`.
- Команда `make sources`.
- Запуск проверки источников в GitHub Actions.
- Проверка источников перед `make xlsx`.
- Обновление README, индекса проекта и инструкции по XLSX.

## Практический смысл

Сборка XLSX больше не зависит только от молчаливого пропуска отсутствующих файлов. Обязательные CSV-источники проверяются до сборки.

## Следующий шаг

Запустить `make preflight` или workflow GitHub Actions и убедиться, что все источники XLSX присутствуют.

# Этап очереди предложений и исправлений

## Добавлено

- Справочник статусов заявок на изменение.
- Рабочая очередь предложений и исправлений.
- Скрипт генерации отчёта по очереди.
- Markdown и CSV отчёты по заявкам.
- Документация по процессу обработки предложений.

## Практический смысл

Менеджер получает контролируемый список предложений сотрудников: что исправить, кто отвечает, какой приоритет и какой статус.

## Ограничение

Приватные данные не должны попадать в публичный репозиторий. Для таких заявок используется статус `NEEDS_PRIVATE_DATA`.

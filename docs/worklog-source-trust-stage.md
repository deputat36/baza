# Этап уровней доверия источников

## Добавлено

- CSV-справочник уровней доверия источников.
- Генератор отчета по уровням доверия.
- Документация по использованию уровней доверия.
- Цель `make source-trust`.
- Сборка отчета в GitHub Actions и GitHub Pages workflow.
- Отчет в пакете запуска, preflight-сводке и стартовом индексе артефактов.
- Ежемесячная задача в регламенте эксплуатации.

## Практический смысл

Сотрудники должны отличать проверенные рабочие данные от черновиков, архивных записей и закрытых данных. Это снижает риск использовать неподтвержденный контакт или внутреннюю информацию как публичную.

## Команда

```bash
make source-trust
```

## Результаты

# Этап UI-карты подсказок

## Добавлено

- Словарь зон будущего интерфейса подсказок.
- Черновая карта размещения подсказок по правилам.
- Валидатор карты UI.
- JSON, Markdown и CSV отчёты по карте.

## Практический смысл

Будущая интеграция сможет понимать не только какую подсказку показать, но и где её показывать в интерфейсе сделки.

## Важно

Проект Навигатора сделок не менялся.

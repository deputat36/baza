# Поиск и теги

## Зачем нужны ключевые слова

Сотрудники ищут информацию не по официальным названиям, а по своим рабочим формулировкам.

Например, один сотрудник ищет `выписка`, другой — `ЕГРН`, третий — `проверить квартиру`.

Поэтому у каждой важной записи должны быть понятные ключевые слова.

## Правило заполнения ключевых слов

Ключевые слова должны включать:

- официальное название;
- разговорное название;
- частую ошибочную формулировку;
- ситуацию, в которой запись нужна;
- связанные документы.

## Примеры

### Выписка ЕГРН

Ключевые слова:

- ЕГРН;
- выписка;
- проверить квартиру;
- собственник;
- обременение;
- кадастровый номер;
- Росреестр.

### Опека

Ключевые слова:

- дети;
- детские доли;
- несовершеннолетние;
- разрешение опеки;
- продажа с детьми;
- доли детям.

### Доверенность

Ключевые слова:

- доверенность;
- представитель;
- продавец в другом городе;
- нотариус;
- право подписи;
- получить деньги.

## Как использовать в Google Таблице

Можно добавить отдельную колонку `Ключевые слова`.

Сотрудник будет искать через обычный поиск по таблице.

## Что добавлять в первую очередь

Ключевые слова нужны для записей с приоритетом 4 и 5.

Для редких записей ключевые слова можно добавлять позже.

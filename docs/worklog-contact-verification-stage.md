# Этап проверки актуальности контактов

## Добавлено

- CSV-регламент проверки контактов.
- Генератор отчета по актуализации контактов.
- Документация по регламенту.
- Цель `make contact-verification`.
- Сборка отчета в GitHub Actions и GitHub Pages workflow.
- Отчет в пакете запуска, preflight-сводке и стартовом индексе артефактов.
- Ежемесячная задача в регламенте эксплуатации.

## Практический смысл

Закрытая Google Таблица с рабочими контактами не должна устаревать. Публичный репозиторий теперь хранит не сами контакты, а порядок проверки: кто отвечает, как часто проверять и где фиксировать результат.

## Команда

```bash
make contact-verification
```

## Результаты

- `build/contact-verification-report.md`
- `build/contact-verification-report.csv`

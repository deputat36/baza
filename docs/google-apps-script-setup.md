# Настройка Google Таблицы через Apps Script

## Назначение

В репозитории есть два безопасных шаблона Apps Script для закрытой Google Таблицы базы знаний.

Они не импортируют реальные рабочие контакты и не содержат приватных данных.

## Базовая настройка

Основной файл:

```text
scripts/google-apps-script/setup-baza.gs
```

Он создаёт базовые листы, закрепляет первую строку, включает фильтры и добавляет стартовое содержание на лист `Главная`.

Запускаемая функция:

```text
setupBazaKnowledgeSheet
```

## Оформление и выпадающие списки

Дополнительный файл:

```text
scripts/google-apps-script/setup_baza_sheet.gs
```

Он применяет:

- цвета вкладок;
- оформление заголовков;
- закрепление первой строки;
- фильтры;
- выпадающие списки для статусов и приоритетов;
- условное форматирование для статусов и приоритетов.

Запускаемая функция:

```text
applyBazaSheetFormattingPlan
```

## Как использовать

1. Создать или открыть закрытую Google Таблицу.
2. Открыть `Расширения -> Apps Script`.
3. Скопировать код из `scripts/google-apps-script/setup-baza.gs`.
4. Запустить `setupBazaKnowledgeSheet`.
5. Скопировать код из `scripts/google-apps-script/setup_baza_sheet.gs`.
6. Запустить `applyBazaSheetFormattingPlan`.
7. Вернуться в таблицу и проверить листы, фильтры, выпадающие списки и цвета.

## Перед запуском

Желательно собрать планы:

```bash
make tabs-plan
make validation-plan
make formatting-plan
```

После этого сверить:

```text
build/google-sheet-tabs-plan.md
build/google-sheet-validation-plan.md
build/google-sheet-formatting-plan.md
```

## Что скрипты не делают

- Не импортируют реальные контакты.
- Не проверяют юридическую точность данных.
- Не обновляют сведения из внешних источников.
- Не заменяют ручную проверку закрытой таблицы.

## Безопасность

Реальные телефоны, ФИО сотрудников, внутренние контакты, условия и комментарии по сделкам должны добавляться только в закрытую Google Таблицу.

В публичный репозиторий эти данные не переносятся.

# Настройка Google Таблицы через Apps Script

## Назначение

Скрипт помогает быстро подготовить рабочую Google Таблицу для базы знаний.

Он создаёт базовые листы, закрепляет первую строку, включает фильтры и добавляет стартовое содержание на лист `Главная`.

## Где находится скрипт

`script/google-apps-script/setup-baza.gs`

Фактический путь в репозитории:

`scripts/google-apps-script/setup-baza.gs`

## Как использовать

1. Создать новую закрытую Google Таблицу.
2. Открыть `Расширения` → `Apps Script`.
3. Создать файл скрипта.
4. Скопировать код из `scripts/google-apps-script/setup-baza.gs`.
5. Сохранить проект.
6. Запустить функцию `setupBazaKnowledgeSheet`.
7. Разрешить выполнение скрипта.
8. Вернуться в таблицу и проверить созданные листы.

## Что создаётся

Скрипт создаёт листы:

- Главная;
- Контакты;
- Ситуации;
- База знаний;
- FAQ;
- Практика;
- Предложения изменений;
- Справочники;
- Синонимы;
- Новостройки;
- Подрядчики;
- Обучение;
- Метрики.

## Что скрипт не делает

Скрипт не импортирует реальные контакты и не заполняет внутренние данные.

Это сделано специально: рабочие контакты, телефоны, ФИО и закрытые условия должны добавляться только в закрытую таблицу и только после проверки.

## После запуска

После запуска нужно:

1. Импортировать CSV по манифесту.
2. Настроить выпадающие списки.
3. Проверить статусы.
4. Защитить справочники и системные поля.
5. Дать доступ тестовой группе.
6. Провести первый пользовательский тест.

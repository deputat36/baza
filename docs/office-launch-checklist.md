# Чек-лист запуска базы в офисе

## Назначение

Чек-лист фиксирует, что нужно сделать перед запуском базы знаний для сотрудников офиса Этажи в Борисоглебске.

Он нужен, чтобы отделить техническую готовность проекта от фактической готовности к использованию в офисе.

## Основной файл

`data/drafts/office-launch-checklist.csv`

## Отчет

```bash
python scripts/tools/build_office_launch_checklist_report.py
```

Результат:

- `build/office-launch-checklist-report.md`
- `build/office-launch-checklist-report.csv`

## Как использовать

1. Проверить задачи со статусом `CHECK`.
2. Закрыть задачи, где `Блокирует запуск = yes`.
3. Внести реальные рабочие контакты только в закрытую Google Таблицу.
4. Провести тест на 2-3 сотрудниках.
5. После исправления критичных замечаний запускать базу на офис.

## Важно

Этот репозиторий остается публично безопасной заготовкой.

Закрытые контакты, внутренние комментарии и данные клиентов не должны попадать в репозиторий.

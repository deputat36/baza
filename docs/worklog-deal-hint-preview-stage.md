# Этап предпросмотра подсказок сделки

## Добавлено

- Безопасные тестовые сценарии подсказок.
- Генератор предпросмотра срабатывания правил.
- JSON, Markdown и CSV-артефакты предпросмотра.
- Документация по предпросмотру.

## Принцип

Навигатор сделок не меняется. Проверка выполняется только на фиктивных сценариях без персональных данных.

# Проверка источников XLSX

## Назначение

Скрипт `scripts/tools/validate_workbook_sources.py` проверяет, что все CSV-файлы, указанные в `scripts/tools/workbook_config.py`, существуют и читаются.

Это нужно, потому что обычная проверка CSV смотрит файлы в папках, но не знает, какие именно источники обязательны для сборки рабочей книги.

## Почему проверка лёгкая

Проверка источников не зависит от `openpyxl` и не создаёт Excel-файл.

Она использует только стандартную библиотеку Python и общую карту источников из `workbook_config.py`.

## Запуск

```bash
python scripts/tools/validate_workbook_sources.py
```

Через Makefile:

```bash
make sources
```

## Что проверяется

- файл существует;
- путь указывает на файл;
- CSV читается;
- файл не пустой;
- первая строка похожа на заголовок.

## Когда проверка падает

Проверка возвращает код `1`, если обязательный источник XLSX отсутствует, пустой или не читается.

## Что делать при ошибке

1. Проверить путь в `scripts/tools/workbook_config.py`.
2. Восстановить отсутствующий CSV или исправить путь.
3. Повторить `make sources`.
4. После успешной проверки запускать `make xlsx` или `make preflight`.

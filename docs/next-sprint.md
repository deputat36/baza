# Следующий спринт

## Цель

Подготовить базу к первому использованию в Google Таблице.

## Задачи

1. Проверить черновик раздела государственных органов.
2. Заполнить реальные контакты в приватной рабочей таблице.
3. Назначить ответственных за разделы.
4. Импортировать первые CSV-файлы.
5. Настроить фильтры и выпадающие списки.
6. Дать доступ небольшой группе сотрудников.
7. Собрать первые практические заметки.

## Первый тест

Проверить, может ли сотрудник быстро найти:

- нужную организацию;
- нужный документ;
- краткий порядок действий;
- частые ошибки;
- связанный контакт.

## Критерий успеха

Сотрудник находит базовый ответ без дополнительного объяснения от менеджера.

# Отчёт по незаполненным данным

## Назначение

Отчёт помогает перед переносом данных в закрытую Google Таблицу увидеть, где в CSV-файлах есть пустые важные поля.

Генератор:

```text
scripts/tools/build_missing_values_report.py
```

Результаты:

```text
build/missing-values-report.md
build/missing-values-report.csv
```

## Запуск

Через Makefile:

```bash
make missing
```

Или напрямую:

```bash
python scripts/tools/build_missing_values_report.py
```

В составе полной проверки:

```bash
make preflight
```

## Что показывает отчёт

- Количество строк данных.
- Количество колонок.
- Количество пустых ячеек.
- Процент пустых ячеек.
- Количество строк с пустым ID.
- Количество строк с пустым статусом.
- Количество строк с пустым приоритетом.
- Примечания по отсутствующим колонкам.

## Как использовать

1. Сначала исправлять строки с пустым ID.
2. Затем проверять пустые статусы.
3. Затем проверять пустые приоритеты.
4. После этого смотреть файлы с высоким процентом пустых ячеек.
5. Реальные рабочие данные добавлять только в закрытую Google Таблицу.

## Важно

Пустая ячейка не всегда ошибка.

Некоторые поля могут быть необязательными или должны заполняться только в закрытой рабочей таблице.

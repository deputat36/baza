# Этап примеров payload подсказок

## Добавлено

- Генератор безопасных примеров request/response для будущих подсказок сделки.
- JSON, Markdown и CSV артефакты с примерами.
- Подключение генератора к Makefile, GitHub Actions, Pages-сборке и реестру контрактов.

## Практический смысл

Будущий интерфейс сможет ориентироваться не только на правила, но и на готовые примеры обмена данными.

## Важно

Проект Навигатора сделок не менялся.

# Этап проверки связей

## Добавлено

- Валидатор `knowledge-index.json`.
- Генератор отчёта по связям базы знаний.
- Документация по отчёту связей.
- Подключение проверок связей к preflight, GitHub Actions и Pages.

## Практический смысл

Проект получает контроль связей между ID. Это снижает риск, что будущий интерфейс карточки сделки не найдёт нужные документы, контакты или ситуации.

# Этап пакета запуска

## Добавлено

- Генератор единого пакета запуска.
- Инструкция по пакету запуска.

## Практический смысл

Менеджер получает один файл с порядком проверки всех ключевых артефактов перед запуском базы знаний в офисе.

## Следующий шаг

Собрать пакет запуска через `make preflight` и использовать его как финальную карту проверки перед переносом рабочих данных в закрытую Google Таблицу.

# Этап видимости данных

## Добавлено

- Словарь видимости данных будущей интеграции.
- Отчёт по публично безопасным и закрытым данным.
- Проверка, что приватные данные не назначены для публичного репозитория.
- Подключение отчёта к preflight, GitHub Actions и Pages-сборке.

## Практический смысл

Перед будущей интеграцией будет понятно, какие данные можно держать в публичном проекте, а какие должны оставаться в закрытой Google Таблице или приватной базе.

## Важно

Проект Навигатора сделок не менялся.

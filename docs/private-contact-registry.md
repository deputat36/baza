# Закрытый реестр контактов

## Назначение

Закрытый реестр контактов нужен для рабочих телефонов, кураторов центрального офиса, городских служб, подрядчиков, нотариусов, кадастровых инженеров и других полезных контактов офиса Борисоглебска.

Публичный репозиторий хранит только структуру полей.

## Исходная схема

`data/drafts/private-contact-registry-schema.csv`

## Генератор отчета

`scripts/tools/build_private_contact_schema_report.py`

## Результаты

- `build/private-contact-schema-report.md`
- `build/private-contact-schema-report.csv`

## Запуск

```bash
make private-contact-schema
```

Или в составе полной проверки:

```bash
make preflight
```

## Что вести только в закрытой Google Таблице

- ФИО кураторов и представителей департаментов.
- Рабочие телефоны.
- Рабочие почты.
- Мессенджеры и внутренние чаты.
- Замещающие контакты.
- Комментарии по качеству работы людей и подрядчиков.
- Закрытые источники и внутренние правила эскалации.

## Что можно хранить публично

- Название направления без ФИО.
- Тип услуги.
- Роль получателя без персональных данных.
- Статус записи.
- Дату проверки.
- Общие правила заполнения.

## Правило

Если есть сомнение, поле считается закрытым и заполняется только в приватной Google Таблице.

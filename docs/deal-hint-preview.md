# Предпросмотр подсказок сделки

## Назначение

Предпросмотр подсказок сделки проверяет, какие правила сработают на безопасных тестовых сценариях.

Это не интеграция с Навигатором сделок. Скрипт не читает реальные сделки и не обращается к CRM.

## Источник сценариев

```text
data/drafts/deal-hint-scenarios.csv
```

## Генератор

```text
scripts/tools/build_deal_hint_preview.py
```

## Артефакты

```text
build/deal-hint-preview.json
build/deal-hint-preview.md
build/deal-hint-preview.csv
```

## Команда

```bash
make deal-hint-preview
```

Или полный набор:

```bash
make preflight
```

## Что проверяется

- какие сигналы есть в тестовом сценарии;
- какие правила должны сработать;
- какие правила реально сработали;
- нет ли неожиданных или отсутствующих правил.

## Практический смысл

Так можно заранее тестировать будущую логику подсказок без доступа к реальным сделкам и без изменения Навигатора.

# Этап метрик использования

## Добавлено

- CSV-шаблон журнала метрик использования.
- Генератор отчета по использованию базы.
- Документация по метрикам использования.
- Команда `make usage-metrics`.
- Сборка отчета в `make preflight`.
- Сборка отчета в GitHub Actions и Pages preview.
- Отчет добавлен в пакет запуска, preflight-сводку, HTML-индекс артефактов и README.
- После запуска офисный чек-лист ссылается на отчет `build/usage-metrics-report.md`.

## Практический смысл

После запуска менеджер сможет видеть, пользуются ли сотрудники базой, где поиск не помогает и какие роли требуют дополнительного обучения.

## Следующий шаг

Собрать отчет через `make usage-metrics` или `make preflight`, затем переносить реальные метрики только из закрытой Google Таблицы или внутреннего инструмента.

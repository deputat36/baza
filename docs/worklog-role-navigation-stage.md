# Этап навигации по ролям

## Добавлено

- Карта навигации по ролям сотрудников.
- Скрипт генерации отчёта по ролевым сценариям.
- Markdown и CSV отчёты по навигации.
- Документация по использованию ролевой навигации.
- Подключение отчёта к preflight, GitHub Actions, Pages-сборке, XLSX и индексам артефактов.

## Практический смысл

Сотрудник видит не все файлы проекта сразу, а понятный маршрут: с чего начать, куда смотреть и какое следующее действие выполнить.

## Ограничение

Навигация не выдаёт доступы к закрытым данным. Она только описывает безопасные маршруты просмотра публичных и сборочных материалов.

# Пакет запуска базы знаний

## Назначение

Пакет запуска — это один управленческий файл, который собирает ключевые отчеты перед запуском базы знаний в офисе.

Он нужен, чтобы менеджер не открывал десятки файлов вручную и проверял запуск в правильном порядке.

## Команда

```bash
python scripts/tools/build_office_launch_packet.py
```

Или через общий сценарий:

```bash
make preflight
```

## Результат

- `build/office-launch-packet.md`
- `build/office-launch-packet.csv`

## Что входит

- стартовая страница артефактов;
- XLSX-книга;
- HTML-превью;
- управленческая сводка;
- готовность разделов;
- ответственность за разделы;
- приемочные сценарии;
- обучение;
- чек-лист запуска;
- регламент эксплуатации;
- планы Google Таблицы;
- интеграционные отчеты;
- preflight-сводка.

## Важно

Пакет запуска не заменяет ручную проверку смысла данных.

Реальные контакты, ФИО сотрудников, клиентские данные и внутренние комментарии должны храниться только в закрытой Google Таблице.

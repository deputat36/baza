# Отчёт по схемам CSV

## Назначение

Скрипт `scripts/tools/build_schema_report.py` собирает заголовки CSV-файлов и формирует отчёт по структуре данных.

Он помогает увидеть:

- какие колонки есть в каждом CSV;
- сколько разных схем используется;
- какие файлы имеют одинаковую структуру;
- где есть пустые или повторяющиеся заголовки;
- какие CSV пустые или не читаются.

## Запуск

```bash
python scripts/tools/build_schema_report.py
```

Через Makefile:

```bash
make schemas
```

## Результат

```text
build/schema-report.md
```

## Как использовать

Перед импортом в закрытую Google Таблицу нужно проверить, что однотипные файлы имеют ожидаемые заголовки.

Разные схемы допустимы, если они соответствуют назначению разделов.

## Ограничение

Отчёт диагностический и не блокирует workflow сам по себе.

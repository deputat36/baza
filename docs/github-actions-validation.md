# Проверка данных через GitHub Actions

Workflow находится здесь:

`.github/workflows/validate-data.yml`

Он запускает:

- проверку CSV-файлов;
- сборку промежуточного XLSX.

Проверка выполняется при изменениях в ветке `main` и может запускаться вручную.

Если проверка падает, нужно открыть лог, найти проблемный CSV, исправить данные и повторить запуск.

Важно: эта проверка техническая. Она не подтверждает актуальность контактов и юридическую точность данных.

# Этап проверки закрытой Google Таблицы

## Добавлено

- Apps Script для технической проверки закрытой Google Таблицы.
- Отдельная инструкция по запуску проверки.
- Обновление общей инструкции по Apps Script.
- Обновление индекса проекта.
- Обновление чек-листа запуска Google Таблицы.

## Файлы

```text
scripts/google-apps-script/validate-baza-sheet.gs
docs/google-apps-script-validation.md
docs/google-apps-script-setup.md

```

## Что проверяет скрипт

- Пустые листы.
- Листы без строк данных.
- Пустые заголовки.
- Дубли заголовков.
- Пустой ID при наличии колонки `ID`.
- Недопустимые статусы.
- Недопустимые приоритеты.

## Результат

В закрытой Google Таблице создаётся или обновляется лист:

```text
Проверка данных
```

## Следующий шаг

После импорта CSV запустить `validateBazaKnowledgeSheet`, исправить замечания и повторить проверку перед тестом сотрудников.

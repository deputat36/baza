# Этап регламента эксплуатации

## Добавлено

- CSV с регулярными задачами эксплуатации базы.
- Генератор отчета по регламенту эксплуатации.
- Инструкция по использованию регламента.

## Практический смысл

После запуска база должна не просто существовать, а регулярно обслуживаться.

Регламент задает ежедневные, еженедельные, ежемесячные и ежеквартальные проверки.

## Следующий шаг

Собрать отчет, перенести регулярные задачи в закрытую рабочую таблицу и назначить реальных ответственных.

# Менеджерский контроль базы знаний

## Зачем менеджеру смотреть базу

База знаний показывает не только справочную информацию, но и слабые места в работе сотрудников.

Если по одной теме часто задают вопросы, значит её нужно усилить в обучении.

## Что контролировать

### 1. Заполненность разделов

Показатели:

- всего записей;
- проверенных записей;
- записей на проверке;
- черновиков;
- устаревших записей.

### 2. Критичные разделы

Особый контроль нужен для разделов:

- документы;
- сделки с детьми;
- ипотека;
- нотариусы;
- банки;
- ЖКХ;
- передача объекта.

### 3. Активность сотрудников

Показатели:

- кто предложил правки;
- кто добавил практический совет;
- какие правки приняты;
- какие предложения отклонены.

### 4. Частые вопросы

Если сотрудники часто ищут одну тему, нужно:

- улучшить запись;
- добавить ключевые слова;
- провести короткое обучение;
- сделать чек-лист.

### 5. Качество партнёров

По партнёрам важно смотреть:

- скорость ответа;
- качество работы;
- пунктуальность;
- цену;
- можно ли рекомендовать.

## Минимальный дашборд в Google Таблице

На листе `Главная` можно вывести:

- количество записей всего;
- количество записей со статусом `Проверено`;
- количество записей со статусом `Нужно проверить`;
- количество новых предложений;
- топ-10 частых ситуаций;
- последние изменения.

## Управленческий вывод

Если база используется, сотрудники начинают меньше задавать повторяющиеся вопросы и быстрее подключают нужных специалистов.

Если база не используется, значит проблема в одном из пунктов:

- неудобный поиск;
- слишком сложная структура;
- нет реальных контактов;
- записи непонятны;
- сотрудники не прошли короткое обучение.

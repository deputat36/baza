# Этап еженедельного дайджеста менеджера

## Добавлено

- Генератор еженедельного дайджеста менеджера.
- Документация по дайджесту.
- Цель `make weekly-digest`.
- Сборка дайджеста в GitHub Actions и GitHub Pages workflow.
- Дайджест в пакете запуска, preflight-сводке и стартовом индексе артефактов.
- Еженедельная задача в регламенте эксплуатации.

## Практический смысл

Менеджер получает один короткий файл, где видно состояние запуска, использования базы, плана улучшений, предложений сотрудников и актуальности данных.

## Команда

```bash
make weekly-digest
```

## Результаты

- `build/weekly-manager-digest.md`
- `build/weekly-manager-digest.csv`

## Следующий шаг

Запустить сборку, проверить дайджест и использовать его как регулярную еженедельную точку контроля.

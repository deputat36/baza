# Этап проверки полей JSON-контрактов

## Добавлено

- Словарь обязательных верхнеуровневых полей JSON-контрактов.
- Валидатор JSON-контрактов.
- Отчёт по найденным и отсутствующим полям.
- Подключение проверки к preflight, GitHub Actions и Pages-сборке.

## Практический смысл

Будущая интеграция сможет проверять не только наличие JSON-файлов, но и минимальную стабильность их структуры.

## Важно

Проект Навигатора сделок не менялся.

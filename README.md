# База знаний специалиста по недвижимости | Этажи Борисоглебск

Проект для создания внутренней базы знаний офиса недвижимости в Борисоглебске.

Цель проекта — собрать в одном месте рабочие контакты, инструкции, документы, алгоритмы сделок, локальную практику и полезные материалы, которые помогают специалистам быстрее и качественнее проводить сделки.

## Быстрый просмотр

Промежуточный вариант проекта можно посмотреть здесь:

- `docs/intermediate-preview.md` — общий обзор проекта, готовые разделы и примеры карточек.
- `docs/google-sheets-mvp.md` — как будет устроена первая Google Таблица.
- `data/drafts/` — черновики данных для будущего импорта.

## Важное правило безопасности

Репозиторий не должен содержать персональные данные клиентов, приватные телефоны сотрудников, внутренние коммерческие условия, доступы, токены, пароли и конфиденциальные документы.

Если в проект будут добавляться рабочие контакты сотрудников, внутренние материалы компании или приватные комментарии, репозиторий необходимо сделать приватным.

## Что хранится в репозитории

- архитектура базы знаний;
- правила заполнения;
- шаблоны таблиц;
- структура будущего CRM-модуля;
- дорожная карта;
- история изменений.

## Где ведутся рабочие данные

На первом этапе рабочие данные удобнее вести в Google Таблице. Репозиторий используется как место хранения структуры, правил и версий проекта.

## Основные разделы

1. Контакты.
2. Государственные органы.
3. Банки.
4. Нотариусы.
5. Кадастровые инженеры.
6. Оценщики.
7. ЖКХ.
8. Управляющие компании.
9. Подрядчики.
10. Документы.
11. Инструкции.
12. Ситуации «Что делать если...». 
13. Практика Борисоглебска.
14. Будущий CRM-модуль.

## Принцип развития

Сначала создаём удобную Google Таблицу и наполняем её данными. Затем, если проект подтвердит пользу для сотрудников, переносим структуру в CRM или отдельный внутренний портал.
